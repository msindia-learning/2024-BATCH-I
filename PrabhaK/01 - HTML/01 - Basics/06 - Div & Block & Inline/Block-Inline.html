Block Element And Inline Element

Block Element

Always Starts on a newline and the browsers automatically add some space before and after the all the elements

Exmaple
  • Address
  • article
  • aside
  • figure
  • figcaption

Inline Elements

Inline Elements Does not Start On a new line.Only Takes up As much width As necessary.
Self Enclosing elements are inline elements are inline elements ,
simply their syntax dont allow them to contain anyother html elememts

Example
  • abbr
  • object
  • strong
  • object
  • label
  • script
