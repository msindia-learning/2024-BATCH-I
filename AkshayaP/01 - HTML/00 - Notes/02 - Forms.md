###HTML FORMS
```
<form>=> form element is used to create a html form.
<input type="">=> input type must displayed in many ways based on its type(radio buttons,checkbox,text fields,submit,button).
<label>=> defines the label for many form elements.
novalidate=> the form should not be validated after submitted.
autocomplete=> autocomplete used to complete the value based on the user enter before.
<select>=> defines the drop down list.
selected=> defines the predefined selection.
textarea=> defines the multiline input field.
button=> defines the clickable button.
fieldset=> used to group related items in a form(create a box around the form).
datalist=> drop downlist of predefined options (just like a select).

```
