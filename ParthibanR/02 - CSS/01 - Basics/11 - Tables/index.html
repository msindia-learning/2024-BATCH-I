HTML Tables

HTML tables allow web developers to arrange data into rows and columns.


Html Table Cells:

Each table cell is defined by a td open and close tag.

Html Table Rows :

Each table row starts with a tr both open and close tag.

Html Table Headers:

Sometimes you want your cells to be table header cells. In those cases use the th both open tag and close tag.
