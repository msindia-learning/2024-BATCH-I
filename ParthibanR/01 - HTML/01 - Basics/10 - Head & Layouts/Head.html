HEAD ELEMENT

The head element is a container for metadata data about data and is placed between the html tag and the tag.HTML metadata is data about the HTML document. Metadata is not displayed.Metadata typically define the document title, character set, styles, scripts, and other meta information.

Head Element:

  1. defines a title in the browser toolbar.
  2. provides a title for the page when it is added to favorites.
  3. displays a title for the page in search engine-results.
  4. accurate and meaningful as possible

LINK ELEMENTS:

The link element defines the relationship between the current document and an external resource. The link tag is most often used to link to external style sheets

META DATA:

The meta element is typically used to specify the character set, page description, keywords, author of the document, and viewport settings. The metadata will not be displayed on the page, but is used by browsers how to display content or reload page, by search engines keywords, and other web services.

Tag Description
