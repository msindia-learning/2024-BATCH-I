### Html video


```html
     
    video:
       The HTML video element is used to show a video on a web page.

    Attribute:
         1.controls:
                 The controls attribute adds video controls, like play, pause, and volume.It is a good idea to always include width and height attributes. If height and width are not set, the page might flicker while the video loads.
           
         2.Autoplay:
                 To start a video automatically, use the autoplay attribute.
           Autoplay muted:
                  Add muted after autoplay to let your video start playing automatically (but muted).

        3.height&width:
                 The input height and width attributes specify the height and width of an elements.

        4.muted:
              The video should be automatically muted.

              
  FileFormat:      Media Type:
   MP4	            video/mp4
   WebM	            video/webm
   Ogg	            video/ogg


   video tags:
        1.video-  Defines a video or movie.used for both open&closing tag.

        2.Source- 	Defines multiple media resources for media elements.

        3.track-  Defines text tracks in media players.
                 

```
